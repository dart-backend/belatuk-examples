# Star Wars

A graphQL application build with Angel 3 framework.

## Build

1. Open a terminal and run:

    ```bash
    dart pub get
    ```

## For Development

1. Open a terminal and run the following:

    ```bash
    dart --enable-vm-service bin/dev.dart
    ```

## For Production

1. Open a terminal and run the following:

    ```bash
    dart bin/prod.dart
    ```
